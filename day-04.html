Sentential Logic: Symbols

Symbols

Name Symbol English Meaning
Conjunction ∧ ‘and’
Disjunction ∨ ‘or’
Conditional → ‘if…then …’
Biconditional ↔︎ ‘if and only if’
Negation ¬ ‘it is not the case that’

Simple Sentences

¬P It is not the case that P
P→Q If P, then Q.
P∧Q Both P and Q.
P∨Q Either P or Q.

English Examples

¬P It is not the case that the pie is ready.
P→Q If the pie is ready, then we eat it.
P∧Q Both the pie is ready and we eat it.
P∨Q Either the pie is ready or we eat it.

Sentential Logic: Syntax

“Syntax”

  • ‘Syntax’ is another word for grammar.
  • It is a set of rules that tell you when a combination of words makes a grammatically well-formed sentence.
  • ‘Dog the of bark’: not a well-formed sentence.
  • ‘Dogs bark’: a well-formed sentence.

Syntax for Natural Languages

  • The syntax of a Natural Language like English is super complicated.
  • (and super neat: there is an entire branch of Linguistics devoted to the study of syntax in natural languages.)
  • It’s amazing that we all recognize grammatical errors, given how complicated the rules are, and we can’t actually say what those rules are.

Syntax for Sentential Logic

  1. Every sentence letter ‘P’, ‘Q’, ‘R’, …, ‘Z’ is a sentence. And so are the sentence letters with subscripts.
  2. If ‘▢’ is a sentence, then ‘¬▢’ is too.
  3. If ‘▢’ and ‘◯’ are both sentences, then:
    1. ‘(▢ ∧ ◯ )’ is also a sentence, and
    2. ‘(▢ ∨ ◯ )’ is also a sentence, and
    3. ‘(▢ → ◯ )’ is also a sentence, and
    4. ‘(▢ ↔︎ ◯ )’ is also a sentence.

Examples of Sentences

(P → Q)

  • ‘P’ and ‘Q’ are sentences, by rule 1.
  • So ‘(P → Q)’ is a sentence, by rule 3c.

Examples of Sentences

¬(P → Q)

  • We just saw that ‘(P → Q)’ is a sentence.
  • So ‘¬(P→Q)’ is a sentence, by rule 2.

Examples of Sentences

¬¬¬¬¬¬¬¬¬Z

  • ‘Z’ is a sentence by rule 1.
  • So ‘¬Z’ is a sentence by rule 2.
  • So ‘¬¬Z’ is a sentence by rule 2.
  • So ‘¬¬¬Z’ is…

Examples of Sentences

((P ∧ Q) ∨ (Q → R))

  • ‘(P ∧ Q)’ is a sentence, by rule 1 and then rule 3a.
  • ‘(Q → R)’ is, by rule 1 and then rule 3c.
  • So ‘((P ∧ Q) ∨ (Q → R))’, by rule 3b.

Examples of Sentences

(((P ∧ Q) ∨ (Q → R))↔︎ ¬¬¬¬¬¬¬¬¬Z)

  • ‘((P ∧ Q) ∨ (Q → R))’ is a sentence.
  • ‘¬¬¬¬¬¬¬¬¬Z’ is a sentence.
  • So, by rule 3d, this is also a sentence.

Not a Sentence

QQQQR

  • ‘Q’ and ‘R’ are sentence letters.
  • But there is no way to combine them, using our rules, to get this result.

Not a Sentence

(P)

  • ‘P’ is a sentence.
  • But there is no rule that lets you surround a sentence with parentheses.

Not a Sentence

¬(P)

  • ‘¬P’ is a sentence.
  • But there is no rule that lets you surround a sentence with parentheses.
  • Specifically, rule 2 does not introduce any parentheses.

“The Main Connective”

  • (P → Q)
  • ¬(P → Q)
  • ¬¬¬¬¬¬¬¬¬Z
  • ((P ∧ Q) ∨ (Q → R))
  • (((P ∧ Q) ∨ (Q → R))↔︎ ¬¬¬¬¬¬¬¬¬Z)

Comparison to Arithmetic

  • → and ↔︎ behave kind of like +
