Direct Derivations

Derivations

  • We call an explicit line of reasoning from some premises to a conclusion a proof.
  • In our formal system, we used derivations to represent proofs.
  • Our system will have three methods of derivation.
  • Unit 4 introduces the first method: Direct Derivation (DD)

Basic Idea

  • To show that an argument is valid, reason step by step, following our rules of inference, from the premises to the conclusion.

An Informal Example

If it rains, streets get wet
If streets get wet, roads are slippery
It rains
Roads are slippery

An Informal Example

  1. Roads are slippery
  2. If it rains, streets get wet (PR)
  3. If streets get wet, roads are slippery (PR)
  4. It rains (PR)
  5. Streets get wet (MP 2,4)
  6. Roads are slippery (MP 3,5)
  7. DD 6

The Same Example, Formally

R → W
W → S
R
S

The Same Example, Formally

  1. S
  2. R → W PR
  3. W → S PR
  4. R PR
  5. W MP 2,4
  6. S MP 3,5
  7. :DD 6
